# Personal Nostr Client

## Current Features

- [x] Home feed
- [x] Discovery Feed
- [ ] Profile management
- [ ] Make post
- [ ] Relay management
- [x] Dark theme
- [ ] NIP-05 support
- [x] Render markdown
- [x] Preview twitter / youtube links
- [ ] Broadcast event
- [ ] Manage followers
- [x] Lighting invoices
- [ ] Image upload
- [ ] Thread view
- [ ] User tipping
- [ ] Reactions

## Setup

```bash
yarn install && yarn start
```
